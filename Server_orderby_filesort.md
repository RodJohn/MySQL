	# 排序算法
	
	所以filesort是否会使用磁盘取决于它操作的数据量大小。
	
	总结来说就是，filesort按排序方式来划分 分为两种：
	
	1.数据量小时，在内存中快排
	2.数据量大时，在内存中分块快排，再在磁盘上将各个块做归并
	
	数据量大的情况下涉及到磁盘io，所以效率会低一些。
	
	根据回表查询的次数，filesort又可以分为两种方式：
	
	1.回表读取两次数据(two-pass)：两次传输排序
	2.回表读取一次数据(single-pass)：单次传输排序
