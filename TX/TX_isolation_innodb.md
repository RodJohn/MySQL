
# innoDB
 
    InnoDB实现了全部的隔离级别
    
# 实现
        
    InnoDB基于MVCC(多版本并发控制协议) 
    将读操作分成 快照读 与 当前读
    快照读
        针对于普通select语句
        通过记录的版本链和ReadView实现了全部的隔离级别
    当前读
        针对于更新语句
        通过两阶段的锁实现了全部的隔离级别
     
         
    
# 优点

    读写操作彼此并不冲突，性能更高  

    innoDB的RR的隔离级别下解决了幻读问题
